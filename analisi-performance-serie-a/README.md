# ⚽ Analisi delle Performance — Serie A 2024/25

> **Progetto Universitario** · Università Politecnica delle Marche

## 📌 Contesto

Analisi comparativa delle performance atletiche della Serie A 2024/25, con l'obiettivo di costruire un ecosistema di Business Intelligence cross-platform che sfrutti più strumenti di visualizzazione per prospettive complementari.

## 🎯 Obiettivo

Creare un ecosistema BI completo per l'analisi delle performance sportive, dimostrando padronanza di molteplici piattaforme di data visualization.

## 🏗️ Approccio

- **Web scraping** per la raccolta automatizzata di dati statistici da fonti online
- Sviluppo di dashboard su **tre piattaforme BI** distinte:
  - **Qlik Sense** — Analisi esplorativa e associativa
  - **Tableau** — Visualizzazioni avanzate e storytelling
  - **Power BI** — Dashboard interattive con modello dati DAX
- Analisi comparativa delle performance di squadre e giocatori
- Confronto delle capabilities e dei punti di forza di ciascuna piattaforma BI

## 🧰 Tech Stack

`Python` · `Web Scraping (BeautifulSoup / Selenium)` · `Qlik Sense` · `Tableau` · `Power BI` · `DAX`

## 📊 Output

- Dataset strutturato delle statistiche Serie A 2024/25
- Tre dashboard interattive (Qlik, Tableau, Power BI)
- Analisi comparativa delle piattaforme BI
