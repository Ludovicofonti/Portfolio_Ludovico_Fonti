# 📈 Previsioni Avanzate con Integrazione di Dati Esterni

> **SYEEW SRL** · 2024 – 2025

## 📌 Contesto

I modelli predittivi esistenti si basavano esclusivamente su dati storici interni (vendite passate, stagionalità), ignorando completamente i fattori esterni che influenzano le performance commerciali. Il risultato erano previsioni fragili di fronte a qualsiasi evento contestuale.

> **Esempio concreto:** un ristorante sul lungomare vede le proprie performance crollare in una giornata di pioggia. Un modello che guarda solo le serie storiche di fatturato non può anticipare questo calo — ma un modello che integra le previsioni meteo sì.

I clienti di SYEEW operavano in settori con dinamiche analoghe: le loro performance erano sistematicamente condizionate da variabili esterne (meteo, traffico, sentiment online, inflazione) che i modelli interni non riuscivano a catturare.

## 🎯 Obiettivo

Migliorare l'accuratezza dei modelli predittivi integrando sistematicamente variabili esogene, rendendo le previsioni più robuste rispetto ai fattori di contesto reali.

## 🏗️ Soluzione

- Identificazione delle **variabili esogene rilevanti** per il settore di ciascun cliente (meteo, traffico, indici di fiducia, dati macroeconomici)
- Integrazione sistematica tramite **API dedicate** per l'ingestion automatica delle fonti esterne
- **Feature engineering** avanzato per trasformare dati grezzi esterni in segnali predittivi utili al modello
- **Hyperparameter tuning** per ottimizzare la performance sui nuovi feature set
- Pipeline di aggiornamento automatico delle variabili esogene in produzione

## 🧰 Tech Stack

`Python` · `Scikit-Learn` · `PyTorch` · `API REST` · `SQL Server` · `Time Series Analysis`

## 📈 Impatto

| Metrica | Risultato |
|---------|-----------|
| Accuratezza modelli predittivi | **+23%** |
| Robustezza previsioni | Significativamente migliorate su fattori di mercato |

## 👤 Ruolo

Sviluppo e ottimizzazione modelli predittivi, integrazione variabili esogene, feature engineering.
